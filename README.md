# OSIM 盃桌球團體賽模擬器

這個專案用來模擬「OSIM 盃桌球團體賽」的排點策略與勝率，協助課堂展示單點思維與多點思維的差異。

## 目前規則

- 賽制：五戰三勝。
- 每隊 7 人。
- 每位球員不可重複上場。
- 五點順序固定為：單、雙、單、雙、單。
- 戰力數字越小代表越強。
- 雙打戰力第一版使用兩人平均值。
- 勝率使用可調 Logistic 模型。

## 開發階段

第一版聚焦在可驗證的勝率模擬器：

- 輸入 A、B 兩隊 7 名球員戰力。
- 調整勝率敏感度。
- 比較貪婪排點與全局最佳排點。
- 匯出 CSV 作為課堂資料紀錄。

目前已擴充成完整對戰展示：

- 可編輯多隊伍資料。
- 顯示所有隊伍彼此對戰的勝率矩陣。
- 顯示平均勝率排名。
- 可切換 Single Scenario / Multi-Scenario。
- 可拖曳配置單一對戰雙方五點排點。
- 可執行 Monte Carlo 多輪模擬並匯出 CSV。

第二版才會往課堂遊戲介面、學生填答流程與資料收集前進。

## 指令

安裝依賴：

```bash
npm install
```

啟動開發伺服器：

```bash
npm run dev
```

執行測試：

```bash
npm test
```

建置 GitHub Pages 用靜態檔：

```bash
npm run build
```
